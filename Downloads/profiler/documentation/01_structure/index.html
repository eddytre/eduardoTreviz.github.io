Line 1 creates the sidebar housing box.
Line 2 creates the sidebar scroll section
 
Lines 3 to 9 create the page shortcuts, the icons at the top. To make one of the icons act like a close toggle, give it the class close-nav
 
Lines 11 to 13 is the page logo. When changing the logo, make sure to replace the width value here and also open the style.css and locate sidebar-logo img and replace the width value there as well. 
 
Line 15 is the page breadcrumb, this is the dark line with the text in it.
 
Line 17 to 23 creates a simple navigaiton item. To make this item selected, give the anchor class embeded in it the class "current" 
 
Line 24 is the sidebar decoration. it's in a separate div because it contains 2 sets of lines for a shadow effect.
 
Lines 25 to 48 is a submenu item. Inside this item you'll find a main anchor class, which has the class "active-submenu" this will make the arrow deploy down. Remove it if your menu is closed by default. Under this anchor class, there is the nav-submenu class that has simple anchor classes for menus inside. To make it open by default, add the class active-submenu to it. To make an element active, give the anchor class within the class "current"
 
From this point on, the navigation elements repeat themselves, the same sturcture applies for the social links bellow, and the final copyright segment is a breadcrumb decoration.

        <a class="portfolio-two-activate"   href="#"><i class="fa fa-th-large"></i></a>
        <a class="portfolio-one-activate active-portfolio-style" href="#"><i class="fa fa-navicon"></i></a>
    </div>
    <div class="decoration"></div>
    <div class="portfolio portfolio-one">
        <div class="portfolio-item">
            <a href="images/general-nature/1.jpg" class="swipebox" title="Caption Goes Here">
                <img class="responsive-image" src="images/general-nature/1.jpg" alt="img">
            </a>
            <h4>Item One</h4>
            <p>
                Consectetur adipiscing elit. Ut sit amet porta dolor, non ipsum.
            </p>
        </div>
    </div>
</div>

This structure can be used to generate the entire portfolio pages. 

The above structure generates the adaptive portfolio. Duplicating the portfolio-item class will create a new portfolio item. By default, the wide portfolio is selected. to change that, move active-portfolio-style to your desired location, and the portfolio portfolio-one must be renamed to either portfolio-two or portfolio-three depending on what you chose.

One Column Portfolio, Two Colum Portfolio, Three Column Portfolio

To create simple portfolios without the adaptive controls, just copy the portfolio portfolio-one class without the portfolio-style class. Replace portfolio-one with portfolio-two ( for two columns ) or portfolio-three ( for three columns)
